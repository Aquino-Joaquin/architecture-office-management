# Architecture Office Management System

This is a full-stack web application developed as a final project for **CENG 307**.

The system is designed to help an architecture and construction office manage:

- Projects
- Clients
- Users and roles
- Office expenses
- Project-related expenses

The application includes a **React frontend** and a **NestJS backend**, with role-based access control.

---

## Technologies Used

- **Frontend:** React
- **Backend:** NestJS
- **Database:** Relational database (via ORM)
- **Authentication:** JWT-based authentication
- **Deployment:** Cloud-based hosting
